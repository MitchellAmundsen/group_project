Welcome to PoliTweets!

Site Overview:

PoliTweets is a site where users can see what others are saying about political candidates, create and vote on various polls, and view beautiful charts with aggregated data.

User Stories (Who, What, Why?)

  • As a registered voter, I want to see an aggregated view of what others are saying on Twitter about a candidate, so that I can be informed.
  • As an opinionated voter, I want to vote on polls for topics that I am passionate about, so that I can voice my opinion.
  • As an undecided voter, I want to view charts of poll results, so that I can make inferences about how the general public feels about an issue.
  • As an avid tweeter, I want to be able to post a tweet on my Twitter from the homepage, so that I can share my views with my friends and invite them to check out Political Twitter.
